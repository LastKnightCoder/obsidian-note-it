此插件仿照语雀小记。

## 使用方法

创建以 `.note` 为扩展名的文件，然后使用 Obsidian 打开即可进行使用。

![](https://cdn.staticaly.com/gh/LastKnightCoder/ImgHosting3@master/image.v6we8nals80.png)

左侧为编辑区（支持 Markdown），右侧为预览区。
